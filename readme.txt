Перед пользованием приложением убедитесь, что у вас установлен Python и все библиотеки, указанные в requirements.txt.
Для установки библиотек напишите в терминале pip install -r requirements.txt

Для корректной работы программы в одном каталоге должны находиться файлы BackgroundMask.py, before_init.py, BossLevel.py, Bullet.py, Buttons.py, Cut-scene.py, EndMenu.py, Infected.py, Instruction.py, load_image.py, main.py, Mask.py, Options.py, Player.py, player_for_boss_level.py, Rating.py, Settings.py, StartMenu.py, Virus.py, rating.csv и Anticoronavirus.exe с подкаталогами data и sounds без изменений внутри. 
!Несоблюдение данного требования не гарантирует корректную работу программы!

Для запуска программы откройте приложение Anticoronavirus.exe.
